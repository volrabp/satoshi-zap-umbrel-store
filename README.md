# Satoshi Zap - Umbrel App Store ⚡

Tento repozitář funguje jako vlastní komunitní repozitář aplikací pro platformu **Umbrel**, který vám umožní nainstalovat peněženku **Satoshi Zap** na váš uzel jedním kliknutím.

---

## 🚀 Jak nainstalovat Satoshi Zap na váš Umbrel

Přidání tohoto repozitáře do vašeho Umbrel uzlu je extrémně jednoduché:

1. Otevřete své webové rozhraní **Umbrel**.
2. Klikněte na aplikaci **App Store** (Obchod s aplikacemi).
3. V pravém horním rohu klikněte na **tři tečky** a zvolte možnost **Add Community App Store** (Přidat komunitní obchod).
4. Vložte odkaz na tento váš GitHub repozitář:
   ```plaintext
   https://github.com/<vas-github-login>/satoshi-zap-umbrel-store
   ```
5. Klikněte na **Add** (Přidat).

Satoshi Zap se okamžitě zobrazí jako krásná nová aplikace přímo ve vašem katalogu připravená k okamžité instalaci! 🎉

---

## 🔒 Konfigurace po spuštění

Po úspěšné instalaci a spuštění peněženky:
1. Otevřete aplikaci (která běží lokálně na vašem Umbrel uzlu).
2. Přejděte do administrace pod cestu `/admin`.
3. Vložte **Admin Key** vaší LNbits peněženky (interní adresa na LNbits uzel v Umbrelu je již přednastavena automaticky).
4. Klikněte na **Uložit** a vaše peněženka i AI asistent jsou okamžitě plně aktivní a synchronizované!
